
Installing and using Jensten Cheque Printer:

 1. Install Python and PIP(preferable latest wersion). PIP can be installed by running "py -m ensurepip --upgrade" in CMD.
 2. Open CMD and navigate to installation folder.
 3. Type "pip install -r requirements txt"
 4. Double click Printer.bat
 5. If webpage does not open automatically, open "http://127.0.0.1:5000" in your browser.
 6. Use CTR + C in CMD to stop the server.
